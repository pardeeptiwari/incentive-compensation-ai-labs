\# Incentive Compensation Analytics – AI Labs Assignment



\## Overview

This repository contains a complete end-to-end analysis of incentive compensation data,

designed to demonstrate data reasoning, business insight generation, and product-ready

feature design for AI-driven compensation platforms.



The dataset is \*\*synthetic but realistic\*\*, modeling 15+ years of incentive outcomes

across roles, regions, plan types, quotas, attainment, payouts, and tenure.



---



\## Repository Structure



data/ → synthetic dataset (50k employee-year rows)

notebooks/ → EDA, charts, insights, feature concepts
